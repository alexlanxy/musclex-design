# Axure 原型项目

这是一个使用 Axure 生成的 HTML 原型项目。

## 在线访问

部署在 GitHub Pages 上，访问地址将会是：
`https://[你的用户名].github.io/[仓库名称]/`

## 本地查看

直接打开 `index.html` 文件即可在浏览器中查看原型。

## 项目结构

- `index.html` - 主页面
- `page_1.html`, `page_2.html` - 其他页面
- `data/` - 数据文件
- `files/` - 页面相关文件
- `images/` - 图片资源
- `plugins/` - 插件文件
- `resources/` - 资源文件（CSS、JS、图片等）

## 更新原型

1. 在 Axure 中更新设计
2. 重新生成 HTML 文件
3. 替换此目录中的文件
4. 提交并推送到 GitHub

```bash
git add .
git commit -m "更新原型"
git push
```
